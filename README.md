# Telegram-content

Черновики и шаблоны в **Markdown** (`.md`): метаданные в YAML frontmatter, текст поста под заголовком «Пост (копировать в Telegram)».

**Как публиковать вручную**

1. Открой `.md` в `channels/…/drafts/` или возьми шаблон из `templates/`.
2. Скопируй раздел **«Пост (копировать в Telegram)»** — от заголовка `## Пост` до конца видимого текста (блоки в тройных обратных кавычках для хэштегов копируй **без** самих кавычек, одной строкой).
3. В клиенте Telegram включи жирный там, где в файле было `**…**`, если вставка не сохранила оформление.
4. Картинки прикрепляй по путям из закомментированных полей `diagram_*` / `media` во frontmatter.

Справочник хэштегов: [`templates/post-tags.md`](templates/post-tags.md).
